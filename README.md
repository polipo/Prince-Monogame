# Prince-Monogame
Monogame implementation of Prince of Persia 2 written in C#, code original idea by Andrea Fallapi.

# Prerequisites

    -Windows
    MonoGame: 3.4 or higher
    OpenTK 1.1
    
    -Linux
    OpenGL 3.0
    SDL2
